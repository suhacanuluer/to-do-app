# To-Do App
This is an API for to-do application. This project developed using Spring Boot.

## Running
To run the API locally you will need to start this commands.

```
cd todo-app-api
mvn spring-boot:run
```

### Technology Stack
- Java 17
- Spring Boot
- Spring Data
- H2 Database
- Swagger
- Maven
